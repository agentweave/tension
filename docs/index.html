Claude Code Plugin

A dial for agent autonomy.

Four profiles control how your AI coding agent behaves — from asking before every action to full self-directed ownership.

$ claude plugin install agentweave/tension

Four profiles, one dial.

Pick the level of autonomy that fits your task. Switch anytime.

Tight
High tension
You drive
Claude asks before acting, explains its reasoning, and waits for your call on every write operation.
Default
Normal
Co-pilot
Standard Claude Code behavior. No overrides. A reset button for when you want baseline back.
Loose
Low tension
You review
Claude drives the task, fixes adjacent issues, auto-commits, and summarizes when done.
Free
No tension
Full ownership
Claude self-directs with a goals checklist, self-checks for drift, and iterates until truly done.

How it works

No hooks, no enforcement, no config files. Just clear behavioral guidelines in markdown.

01

Set a profile

Run the slash command to write behavioral instructions to .tension.md

/tension:set loose
02

Claude reads it

The plugin wires CLAUDE.local.md to reference your profile. Claude picks it up automatically.

03

Behavior changes

Claude follows the profile's guidelines — communication style, error handling, commit strategy, scope control.

Usage

Two commands. That's it.

/tension:set tight # You drive, Claude assists
/tension:set default # Standard behavior
/tension:set loose # Claude drives, you review
/tension:set free # Full ownership
/tension:status # Show active profile
Pure markdown — no code to build
No hooks or enforcement
Edit profiles by hand if you want
Works with any permission mode
